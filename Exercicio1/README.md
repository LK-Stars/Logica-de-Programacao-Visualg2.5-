# Exercício 1 - Cálculo de Média Escolar

## 📌 Objetivo
Desenvolver um algoritmo que receba o nome de um aluno e suas notas de três bimestres, realizando o cálculo da média final.

## 🧠 Conceitos utilizados
- Entrada de dados (leia)
- Variáveis
- Operações matemáticas
- Cálculo de média aritmética
- Saída de dados (escreval)

## 💻 Descrição do problema
O algoritmo solicita ao usuário o nome do aluno e as notas do primeiro, segundo e terceiro bimestre. Em seguida, calcula a média das três notas.

Ao final, exibe a seguinte mensagem:

> O aluno (nome) teve como média a nota (média)

## 🚀 Resultado esperado
O programa deve ser capaz de:
- Receber corretamente o nome do aluno
- Receber três notas numéricas
- Calcular a média aritmética
- Exibir o resultado formatado com nome e média

## 🎯 Aprendizado
Este exercício reforça o uso de variáveis, entrada de dados e operações matemáticas básicas em lógica de programação.
