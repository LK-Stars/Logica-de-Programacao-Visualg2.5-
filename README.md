## 🧠 Conteúdos Abordados

Os exercícios envolvem conceitos fundamentais como:

* Entrada e saída de dados
* Estruturas condicionais (`se`, `senao`)
* Estruturas de repetição (`enquanto`, `para`)
* Variáveis e tipos de dados
* Operações matemáticas
* Lógica de decisão

## 🛠️ Ferramenta Utilizada

* **Visualg 2.5** — Ambiente de simulação de algoritmos com pseudocódigo em português.

## ▶️ Como Executar

1. Abra o **Visualg 2.5**
2. Clique em **Arquivo > Abrir**
3. Selecione o arquivo `.alg` desejado
4. Pressione **F9** ou clique em **Executar**
